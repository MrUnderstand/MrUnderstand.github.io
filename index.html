Makefile

  • 用于大型工程的编译，Makefile文件指定如何编译，Make是一个命令工具，用于解析Makefile。编译，源文件生成目标文件，将目标文件合成可执行文件，叫做链接（link），主要是链接函数和全局变量。由于目标文件太多，链接时需要指定目标文件名，比较麻烦，因此就把目标文件打包，生成.a文件（Unix，Archive file）（Windows，.lib文件，library file）。主要用于C/C++工程编译。

  • 格式：

    target : prerequisites

    ​ command:

    target：目标文件，可执行文件，标签（伪目标）

    prerequisites：生成该target所依赖的文件

    command：前面需要有个tab键，shell命令

  • Make工作方式

    1. make在当前目录下寻找“Makefile”或“makefile”的文件，可以用 -f 指定文件。
    2. 找文件中的第一个目标文件，作为最终的目标文件。
    3. 目标文件不存在或者文件存在，但其依赖的文件修改日期比文件新，则会执行后面的命令来生成目标文件。
    4. 如果该目标文件所依赖的文件也不存在，则make会在当前的文件中寻找该文件的依赖性。

  • Makefile

    Makefile里面主要有5个部分：显示规则，隐式规则，变量定义，文件指示，注释。

    1. 显示规则：显示规则定义如何生成一个目标文件
    2. 隐式规则：make的自动推到功能
    3. 变量定义：类似于C语言中的宏定义功能
    4. 文件指示：
      1. 文件包含： include
        1. make命令执行时，会把include 的文件的内容放在当前的位置。
      2. 预编译：类似于C语言的 #if
      3. 多行的命令
    5. 注释：#

    最终会这样执行：

    1. 读入Makefile文件
    2. 读入include的文件
    3. 初始化变量
    4. 推到隐式规则
    5. 根据规则生成目标文件
